# SorceryMongoid

Mongoid adapter for sorcery.

## Installation

Add this line to your application's Gemfile:

    gem 'sorcery_mongoid'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sorcery_mongoid
